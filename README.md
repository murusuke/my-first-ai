# my-first-ai

こんにちは！👋  
ここは、AI（えーあい）をつかって「はじめてのものづくり」をするためのれんしゅうばこです。

## このリポジトリでやること
- 7日かん、まいにち少しずつさわってみる
- ちいさくても「1つできた！」をつくる
- わからなくても、ためして、なおして、またためす

## AIってなに？
AIは、
- しつもんにこたえてくれたり
- アイデアをいっしょにかんがえたり
- ぶんしょうをなおすのをてつだってくれたりする

「べんりなおてつだいロボット」みたいなものです。🤖

## つかいかた（かんたん3ステップ）
1. やってみたいことを1つきめる
2. AIに「どうやるの？」ときく
3. できたら、つぎはもうすこしだけむずかしいことにちょうせんする

## だいじなやくそく
- まちがえてもOK！
- できない日は、1行だけでもすすめばOK！
- 「できた！」をちゃんとほめる
